## 🌐 Project Overview

ST-X-0 (part of the Surya Series) is built to address the complexities of modern linguistic interaction, specifically focusing on the nuances of **Hinglish** (Hindi-English) code-switching. Whether it's complex logical deduction or processing massive documents, ST-X-0 provides a robust foundation for developers and researchers.

### 💎 Key Highlights
* **Model Series**: ST-X-0 (Surya V1.5).
* **Context Window**: Supports a massive **150,000+ Tokens** for long-context memory.
* **Architecture**: Optimized Transformer-based Causal LLM with advanced reasoning capabilities.
* **Primary Focus**: Deep logical reasoning, natural Hinglish conversational flow, and metadata manipulation.
